WhatsApp da loja

Use esta área para conectar o número da loja ao WPPConnect, enviar mensagens automáticas e responder consultas simples de pedido no WhatsApp.

Modo recomendado

A Donilla usa o WPPConnect do próprio servidor para enviar mensagens automáticas e responder consultas simples no WhatsApp. Esse é o modo principal desta tela.

O que precisa estar pronto

WPP_SERVER_URL aponta para o WPPConnect, WPP_SESSION_NAME define a sessão, WPP_SECRET_KEY autentica a integração e APP_BASE_URL informa a URL pública deste backend. O webhook interno usado pelo sistema será /whatsapp/webhook.

Passo a passo

  1. Consulte a sessão para ver o estado atual.
  2. Inicie a sessão no WPPConnect.
  3. Busque o QR Code e escaneie no celular da loja.
  4. Ative as mensagens automáticas e envie um teste.
Pausa do bot Bot em funcionamento normal.

Use este botão para pausar o bot inteiro no WhatsApp sem desligar container. Enquanto estiver pausado, a Donilla não responde mensagens recebidas nem dispara automações de pedido/status.

O teste manual continua disponível para validar a integração.

QR Code para parear Leia com o celular da loja

Quando o QR estiver visível, abra o WhatsApp do número da loja, entre em aparelhos conectados e escaneie o código para concluir a conexão.

Integração externa opcional

Use esta opção apenas se você não quiser usar o WPPConnect interno da Donilla. Nesse modo, a Donilla só repassa os dados do pedido para outro sistema seu, e esse sistema externo assume o envio das mensagens.

Exemplo: você já tem outro bot, outro servidor ou outra automação e quer que a Donilla envie um webhook para ele sempre que um pedido for criado ou atualizado.

Variáveis disponíveis nas mensagens: {cliente_nome}, {cliente_telefone}, {pedido_id}, {status_entrega_label}, {status_pagamento_label}, {status_anterior_label}, {status_mensagem}, {valor_total}, {valor_entrega}, {metodo_pagamento}, {observacoes}, {endereco_resumo}, {itens_resumo}, {tempo_entrega_min}, {tempo_entrega_max} e {previsao_entrega}. Você pode usar quebras de linha normalmente.
